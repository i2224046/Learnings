PHP Control Structures

条件分岐と繰り返しの使い分け ＆ 「すべてforで書く」裏技チートシート

使い分け早見表

グループ 構文 主な用途・どんな時に使う？
条件分岐 if / else 複雑な条件、範囲指定（>, <）。「もし〇〇なら」という汎用的な分岐。
switch 1つの変数を複数の値と（緩く）比較。状態による処理分け（やや古い書き方）。
match 1つの変数を厳密に比較し、値を返す。switchの現代版（PHP8〜）。
繰り返し foreach 配列の中身を順番に取り出して全て処理したい時（一番使います）。
for 「1から10まで」など、繰り返す回数が明確に決まっている時。
while 条件が真の間繰り返す。いつ終わるか分からない時。
do-while 最低1回は必ず実行し、その後条件を満たす間繰り返す時。
