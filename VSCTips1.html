PHP Emmet Configuration

PHPファイル内でHTMLタグの高速入力（Emmet）を有効にする手順：

  • Ctrl + , (カンマ) を押して設定画面を開きます。
  • 検索バーに emmet include と入力します。
  • Emmet: Include Languages という項目を探し、「項目の追加」をクリックします。
  • 追加された入力欄に以下の値を入力して保存します。
    項目: php
    値: html
